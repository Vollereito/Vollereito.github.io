#Markdown 基本语法
Markdown的语法十分简单，常用的标记符号不超过十个，用于日常写作记录绰绰有余，不到半小时就能完全掌握

Quick Start

标题语法

要创建标题，请在单词或短语前面添加井号 (#) 。# 的数量代表了标题的级别

Heading level 1 一级标题

Heading level 2 二级标题

Heading level 3 三级标题

Heading level 4 四级标题

Heading level 5 五级标题
Heading level 6 六级标题

More info: 标题语法

注：不同的 Markdown 应用程序处理 # 和标题之间的空格方式并不一致。为了兼容考虑，请用一个空格在 # 和标题之间进行分隔。

段落

要创建段落，请使用空白行将一行或多行文本进行分隔。

More info: 段落

换行

在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行(
)。

几乎每个 Markdown 应用程序都支持另一种换行方式：HTML 的
标签。

More info: 换行

强调

**粗体（Bold） –要加粗文本，请在单词或短语的前后各添加两个星号（asterisks）或下划线（underscores） **

斜体（Italic） –单词或短语前后添加一个星号（asterisk）或下划线（underscore）

粗体（Bold）和斜体（Italic） –三个星号或下划线

More info: 强调

引用

###要创建块引用，请在段落前添加一个 > 符号。

代码语法

要将单词或短语表示为代码，请将其包裹在反引号 (`) 中。

`
要创建代码块，请将代码块的每一行缩进至少四个空格或一个制表符。

void
imi_onm_init(struct lib_globals *zg)
{
struct onm_client *oc = NULL;

/* Create ONM client for ELMI */
oc = onm_client_create (zg, 0);
if (!oc)
    return;
printd("[%s:%d]   imi_onm_init  &  onm_client_create \n",__func__, __LINE__);

/* Set version and protocol.  */
onm_client_set_version (oc, ONM_PROTOCOL_VERSION_1);
onm_client_set_protocol (oc, IPI_PROTO_IMI);

/* Set required services. */
//    onm_client_set_service (oc, ONM_SERVICE_DESCRIBTION);

/* ONM message callbacks. */
onm_client_set_callback (oc, ONM_MSG_SERVICE_REPLY, imi_onm_recv_service);
onm_client_set_callback (oc, ONM_MSG_DESCRIBTION, imi_onm_recv_lldp_describtion);

/* Set disconnect handling callback */
onm_client_set_disconnect_callback (oc, imi_onm_disconnect_process);

/* Adding delay to start onm_client to let ecpd learn NSM update first followed
 *by ONM updates; ZebOS00078427
 */
onm_client_start (oc);
}

`

More info: 围栏代码块

分隔线

要创建分隔线，请在单独一行上使用三个或多个星号 (***)、破折号 (—) 或下划线 (___) ，并且不能包含其他内容

***

---

_____________

链接语法

链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。

超链接Markdown语法代码：[超链接显示名](超链接地址 "超链接title")

对应的HTML代码： <a href="超链接地址" title="超链接title">超链接显示名</a>

这是一个链接 [Markdown语法](https://markdown.com.cn)。

渲染效果如下：

这是一个链接 Markdown语法。

More info: 链接语法

图片语法

要添加图像，请使用感叹号 (!), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。

插入图片Markdown语法代码：![图片alt](图片链接 "图片title")。

对应的HTML代码：<img src="图片链接" alt="图片alt" title="图片title">

![这是图片](/assets/img/philly-magic-garden.jpg "Magic Gardens")

转义字符

要显示原本用于格式化 Markdown 文档的字符，请在字符前面添加反斜杠字符 \ 。

\* Without the backslash, this would be a bullet in an unordered list.

渲染效果如下：

* Without the backslash, this would be a bullet in an unordered list.

More info: 转义字符

内嵌 HTML 标签

对于 Markdown 涵盖范围之外的标签，都可以直接在文件里面用 HTML 本身
如需使用 HTML，不需要额外标注这是 HTML 或是 Markdown，只需 HTML 标签添加到 Markdown 文本中即可。

More info: 内嵌 HTML 标签
